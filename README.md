# 📋 Task Tracker — Dual-POV Productivity Dashboard

A sleek, responsive task management dashboard featuring a dual-perspective workflow that connects management oversight with employee execution in a clean visual workspace.

## 🚀 Features
- **Dual Perspective System (POV Switcher):**
  - **Manager Mode:** Complete control to create, assign, distribute, and track the status of all workload items across the team.
  - **Employee Mode:** A focused, streamlined view tailored specifically for the individual worker to view tasks, update progress, and mark items as complete.
- **Visual Status Tracking:** Categorize tasks seamlessly into structured status columns (e.g., To-Do, In Progress, Done) to map out workflows.
- **Smart Filtering & Search:** Quick filter toggles and search capabilities to instantly locate high-priority tasks or specific categories.
- **Modern Dark UI:** A highly scannable, minimal dark-themed dashboard interface engineered to reduce eye strain and maximize productivity focus.

## 🛠️ Built With
- **Frontend Framework:** React + Vite
- **State Management:** React hooks handling active role toggles and distinct data layers
- **Styling:** Custom component-driven CSS layout architectures
